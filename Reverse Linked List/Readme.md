# Reverse Linked List

## Input
- A singly linked list represented by the head node.
- Each node contains:
  - An integer value (`val`).
  - A reference to the next node (`next`).

### Example Input
```plaintext
1 -> 2 -> 3 -> 4 -> 5 -> null
```

## Output
- A new head node representing the reversed linked list.
- The linked list is now in reverse order.

### Example Output
```plaintext
5 -> 4 -> 3 -> 2 -> 1 -> null
```
